Como registrarme ...


Como cliente

Ingrese un codigo de usuario en la casilla: Usuario
Ingrese la contraseña correspondiente a dicho usuario en el casillero: Password
En la lista desplegable de roles, elija: Cliente
seleccione aceptar.


Como operario

Ingrese un codigo de usuario en la casilla: Usuario
Ingrese la contraseña correspondiente a dicho usuario en el casillero: Password
En la lista desplegable de roles, elija: Operario
Ingrese el codigo de confirmacion de operario otorgado por la empresa en la casilla: Codigo de confirmacion,
esta se habilita una vez usted seleccione el rol de operario.

esta se habilita una vez usted seleccione el rol de operario.
